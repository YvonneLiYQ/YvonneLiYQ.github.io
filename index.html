Title2: LOVELY TRANSFER

Project Introduction：

With the increasing of human material level, art has begun to enter People's Daily life more widely. With the development of science and technology, more and more artists focus on the field of AI to explore the combination of technology and art. I chose LovelyTransfer as the Final Year Project, which is an image style transfer web application based on VGG16. This project required front-end and back-end development, as well as finding a suitable dataset to train VGG16 on. I use the React.js framework for front-end development, and Python for machine learning in the algorithm part. In this project, I provided multiple styles of image style transfer buttons to satisfy the user's creativity, and I pretrained the model to improve the responsiveness and usability of the application
